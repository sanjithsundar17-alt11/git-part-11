## Working Directory
This is where files are created and modified.

Example command:
git status

---

## Staging Area
Files are prepared before commit.

Example command:
git add test.txt

---

## Local Repository
Changes are saved in the local repository.

Example command:
git commit -m "first commit"

---

## Remote Repository
Code is uploaded to GitHub.

Example command:
git push origin master
